# 📊 Stock Market Prediction and Portfolio Optimization

## 🚀 Overview

This project is a data-driven financial system that combines **Machine Learning** and **Modern Portfolio Theory (MPT)** to predict stock prices and optimize portfolio allocation. The goal is to help investors maximize returns while minimizing risk using automated, data-driven techniques.

---

## 🎯 Features

* 📈 Stock price prediction using Machine Learning
* 📊 Portfolio optimization using Monte Carlo Simulation
* ⚖️ Risk vs Return analysis (Efficient Frontier)
* 🔗 Correlation matrix for diversification
* 📉 Interactive visualizations using Plotly
* 🌐 User-friendly dashboard with Streamlit

---

## 🧠 Tech Stack

* Python 3.10+
* Pandas, NumPy
* Scikit-learn
* yfinance
* Plotly
* Streamlit

---

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/stock-portfolio-optimizer.git

# Navigate to project folder
cd stock-portfolio-optimizer

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

---

## 📊 How It Works

1. Fetches historical stock data using yfinance
2. Cleans and preprocesses data
3. Creates lag features and rolling statistics
4. Predicts stock prices using ML models
5. Simulates thousands of portfolios
6. Identifies optimal portfolio using Sharpe Ratio

---

## 📈 Output

* Predicted stock prices
* Efficient Frontier graph
* Optimal portfolio allocation
* Risk and return metrics

---

## ⚠️ Limitations

* Depends on historical data
* Cannot predict sudden market events
* Does not include transaction costs or taxes
